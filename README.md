# 🎬 Netflix Data Analysis Dashboard \| Power BI

## 📌 Project Overview

This project analyzes Netflix customer viewing behavior and content
performance using **Power BI**. The objective is to transform raw,
unclean datasets into meaningful business insights through data
cleaning, ETL, visualization, and interactive dashboards.

## 🎯 Problem Statement

Netflix has customer and content datasets containing missing values and
inconsistencies. The goal is to clean, transform, model, and visualize
the data to generate actionable business insights.

## 📂 Dataset

-   **Customer Dataset (20,000 records)**: Name, Age, Gender, Country,
    Subscription, Show ID, Rating, Watch Time
-   **Show Dataset (20,000 records)**: Show ID, Title, Type, Genre,
    Director, Cast, Release Date, Content Rating

## 🛠 Data Cleaning

-   Merged datasets using Show ID
-   Replaced missing directors with **Unknown**
-   Standardized dates
-   Removed duplicates
-   Created Age Group and Release Category columns
-   Built DAX measures and KPIs

## ⚙ ETL

-   Excel preprocessing
-   Power Query transformations
-   Data modeling
-   DAX calculations
-   Interactive dashboard creation

## 📊 Dashboard

1.  Customer Demographics
2.  Subscription Analysis
3.  Top 10 Most Watched Content
4.  Popular Genre by Country
5.  Rating Distribution
6.  Trending Content
7.  Subscription Market Share
8.  High vs Low Rated Genres
9.  Top Directors
10. Country Performance
11. Best Release Month
12. Genre Ratings by Country

## 📈 Key Insights

-   Standard plan dominates subscriptions.
-   60+ age group records the highest watch time.
-   New releases receive higher engagement.
-   Brazil has the highest engagement, while Japan is among the lowest.
-   March is the best month for content releases.
